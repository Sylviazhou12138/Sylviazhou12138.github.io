Homework 5: Altair Visualization

This page contains two visualizations created with Python, Altair, and Vega-Lite.

Plot 1: UFO Sightings per Year

This visualization shows the number of UFO sighting reports aggregated by year. After loading the dataset, I converted the date_time column into a proper datetime format and extracted the year values using Pandas. I then grouped the data by year and counted the number of sightings for each year. This preprocessing step helped produce a cleaner temporal summary suitable for a bar chart. In designing the plot, I used a bar encoding with year on the x-axis and count of reports on the y-axis. I applied a consistent blue color to keep the visualization simple and avoid overwhelming the viewer with unnecessary variation. The chart emphasizes long-term temporal trends and highlights periods with spikes in reported UFO sightings.

Plot 2: Interactive UFO Map

This visualization displays the geographic distribution of UFO sightings using latitude and longitude. I again parsed the date_time into timestamps and extracted the year so that sightings could be filtered by time. Each point represents one reported sighting, and the scatter layout reveals spatial clustering patterns across the United States. To enhance clarity and user engagement, I added an interactive slider parameter that filters the points by year. This interactivity allows viewers to explore how UFO sightings vary across different decades and to observe changes in reporting patterns over time. It also helps reduce overplotting by letting the user focus on a specific temporal window, making spatial patterns easier to see.

For the second plot, I implemented an interactive year slider using an Altair parameter bound to a range input. This enables dynamic filtering of data points based on the selected year. The interactivity improves readability by avoiding clutter, and it allows viewers to explore temporal and spatial relationships in the dataset more effectively.
