Homework 5: Altair Visualization

This page contains two visualizations created with Python, Altair, and Vega-Lite.

Plot 1: UFO Sightings per Year

This bar chart shows the number of UFO sighting reports per year. Each bar represents a year, and the color gradient highlights temporal change using a plasma color scheme. The data was grouped by year after parsing datetime values from the dataset.

Plot 2: Interactive UFO Map

This interactive scatter map displays the geographic locations of UFO sightings. Users can move a year slider to explore sightings in different time periods. The interactivity helps users notice temporal and spatial clustering of reports across the US.
